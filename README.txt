Pixi Energy Final Pro

Что внутри:
- React + Vite
- PixiJS
- image.png
- живая камера
- многослойное пламя
- halo / mist / body energy
- embers
- отдельные вспышки

Запуск:
1. npm install
2. npm run dev

Production build:
1. npm run build
2. папку dist можно деплоить

Для Telegram Mini App:
- собираешь через npm run build
- деплоишь dist
- вставляешь ссылку в WebApp / Mini App
